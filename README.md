# 📌 Regressão Linear com Gradiente Descendente

Este projeto implementa, do zero, um modelo de Regressão Linear Multivariável utilizando Python e NumPy. O objetivo é prever valores (como preços de casas) a partir de múltiplas características. O código foi desenvolvido com foco educacional, mostrando passo a passo como funciona o treinamento de um modelo de Machine Learning sem o uso de bibliotecas prontas.

## ⚙️ Funcionalidades
Manipulação de dados com NumPy, implementação manual da função de previsão (tanto com loop quanto vetorizada), função de custo (Erro Quadrático Médio), cálculo do gradiente e algoritmo de gradiente descendente, além da visualização da convergência com Matplotlib e comparação entre valores previstos e reais.

## 🧪 Dataset
O conjunto de dados contém exemplos com as seguintes features: tamanho da casa, número de quartos, número de andares e idade do imóvel. A saída do modelo é o preço da casa.

## 🚀 Como executar
Instale as dependências com o comando: pip install numpy matplotlib. Certifique-se de que o arquivo de estilo "deeplearning.mplstyle" está presente na mesma pasta do script. Em seguida, execute o código com: python oi.py.

## 📈 Saída esperada
O programa irá exibir as previsões do modelo, compará-las com os valores reais, mostrar a redução do custo ao longo das iterações e gerar um gráfico com a convergência do algoritmo.

## 🎯 Objetivo
Este projeto tem como objetivo reforçar conceitos fundamentais de Machine Learning, gradiente descendente, álgebra linear aplicada e implementação de algoritmos do zero.

## 📚 Tecnologias
Python, NumPy e Matplotlib.
